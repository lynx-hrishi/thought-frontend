# Product Overview

## Project Purpose
ThoughtDrop is a Flutter-based dating and social connection application that focuses on creating meaningful connections through shared thoughts and interests. Despite the repository name "currency_converter", this is a full-featured social matching platform.

## Key Features

### Authentication & User Management
- Email-based authentication with OTP verification
- Session management with persistent cookies
- User profile creation and editing with personal details
- Profile image and post image uploads

### Matching System
- Swipe-based user matching (like/pass functionality)
- Intelligent match recommendations based on preferences
- Partner preference filtering (gender, age range, interests)
- Match status tracking and notifications

### User Profiles
- Comprehensive profile information (name, gender, date of birth, zodiac sign, profession)
- Interest-based matching
- Multiple image support (profile + post images)
- Age-based filtering

### Social Features
- Chat functionality for matched users
- Message history with local storage
- User dashboard with match overview
- Profile editing and customization

### Technical Features
- Cross-platform support (Android, iOS, Web, Windows, Linux, macOS)
- Persistent session management with cookie-based authentication
- Custom image loading with Dio HTTP client
- Responsive design for mobile and web

## Target Users
- Young adults seeking meaningful romantic connections
- Users who value personality and interests over superficial attributes
- People looking for compatibility-based matching
- Users comfortable with mobile-first dating applications

## Use Cases
- Browse potential matches based on preferences
- Like or pass on suggested profiles
- Chat with matched users
- Manage personal profile and preferences
- View match history and conversations
- Update partner preferences and search criteria
